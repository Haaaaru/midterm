偶数、奇数の判定

以下の仕様を満たすように、偶数を判定する関数 isEvenNumber を定義してください。定義は 03.js に記述してください。

記法

      
let label = classifyInt(integer);

引数

  • integer：整数

返り値

"偶数"もしくは"奇数"のどちらか。

integer が偶数の時は前者を、そうでない時は後者を返します。

利用例


classifyInt(13); // "奇数"
classifyInt(2);  // "偶数"

classifyInt(0); // "偶数"

classifyInt(-2); // "偶数"
classifyInt(-3); // "奇数"
